## 开始

### 加速器

由于需要使用 GitHub，为了确保良好的网络环境，可以使用 Watt Toolkit 加速 GitHub，访问[软件官网](https://steampp.net/download)下载。

安装完成后打开，在“网络加速”页面勾选上“GitHub”，然后点击右上角“一键加速”按钮。如果不成功，可在“加速设置”中点击“安装证书”。

### GitHub

你需要先拥有一个 GitHub 账号才能修改文档，在[此处](https://github.com/signup)注册。

### Git

#### 安装

你需要先安装 [Git](https://git-scm.com/install/windows) 才能进行克隆/提交仓库等操作。

#### 配置

>以下教程主要面向 Windows。其中的 Git Bash 命令可以通用。

首先需要先配置 SSH 密钥。

打开文件资源管理器，打开任意文件夹，然后按住Shift右键空白处，点击右键菜单中的“Open Git Bash here”选项。

这时会弹出 Git Bash 窗口，等待提示文本加载完成后在其中输入以下命令：（注意粘贴是`Shift+Ins`，也可以右键选择“Paste”粘贴）

```bash
ssh-keygen -t rsa -C "xxx@xxx.com"
cd ~/.ssh
cat id_rsa.pub
```

其中`xxx@xxx.com`要换成自己 GitHub 账号绑定的邮箱。

登录你的 GitHub 账号，打开[这个链接](https://github.com/settings/ssh/new)创建一个 SSH 密钥。“Title”随意，“Key”粘贴刚才复制的密钥。

在 Git Bash 中输入`ssh -T git@github.com`确认是否成功，如果看到`successfully`说明成功添加了。

然后添加一下个人信息。

在 Git Bash 中输入以下命令：

```bash
git config --global user.email "xxx@xxx.com"
git config --global user.name "xxxx"
```

其中`xxx@xxx.com`要换成自己 GitHub 账号绑定的邮箱，`xxxx`要换成自己的 GitHub 用户名。

## 修改

先找到一个空文件夹，然后按住Shift右键空白处，点击右键菜单中的“Open Git Bash here”选项。

在 Git Bash 窗口中输入一下命令：

```bash
git clone https://github.com/fblaze62/fblaze-docs.git
git pull origin main
cd fblaze-docs
```

然后即可对仓库进行修改（推荐使用[Visual Studio Code](https://code.visualstudio.com/)打开该文件夹进行修改）。文档格式请参考[Teek主题文档](https://vp.teek.top/guide/directory-structure.html)。

修改完成后，在 Git Bash 中输入以下命令提交：

```bash
git commit -m "提交说明"
git push origin main
```

请注意，每次修改前都需要先在 Git Bash 中输入`git pull origin main`拉取一次。

你也可以使用 Visual Studio Code 侧边栏的“源代码管理”的GUI界面拉取或提交更改。
