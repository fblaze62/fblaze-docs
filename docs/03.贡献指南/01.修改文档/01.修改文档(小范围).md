你需要先拥有一个 GitHub 账号才能修改文档，在[此处](https://github.com/signup)注册。

打开要修改的文档，滚动到页面底部，点击“在 GitHub 上编辑此页”按钮，会自动跳转到该文档的 GitHub 页面。

对文档进行修改，完成后点击右上角的“提交更改...”，等待AI自动编写提交信息（也可以自行编写，中英文均可，推荐英文）后点击“提交更改”按钮即可。

此方式仅适用于小范围修改，如需进行更方便的修改请查看[修改文档(克隆仓库)](https://docs.fblaze62.top/guide/contribute/docs/clone.html)部分。
