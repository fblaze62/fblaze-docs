服务器需要安装 **Minecraft 1.21.1 NeoForge** 并添加所需模组才能加入，你只需下载由 FeatherBlaze 制作的整合包“羽焰的建筑”即可。该整合包包含必装的模组、一部分优化模组以及推荐的资源包。

你可以在QQ群“FBlaze Matrix·Players”（右上角图标）的群文件中找到该整合包，或者在 [123云盘](https://1835134308.share.123pan.cn/123pan/PpdmTd-HtTRH?pwd=fmsv#) 下载。

整合包的模组介绍请看[整合包功能](https://docs.fblaze62.top/guide/mods/adventure.html)部分。

***注意***：**请不要随意更新模组**。“信雅互联”使用改版以适配Paper服务端，“输入法冲突修复”经过版本降级以适配旧版“Axiom”，“More Culling”经过拆包并手动升级内置的“MixinSquared”模组以保证与“Flashback”的“MixinSquared”版本匹配，多个机械动力/地毯附属升级后会报错，“Flashback”模组会在进行一些更改后出现位置原因的字体加载失败，“Xaero的地图增强”仅兼容特定版本的“Xaero的小地图”/“Xaero的世界地图”。

### FAQ

**Q1:** 我无法启动整合包！

**A1:** *请手动设置游戏 Java 设置为 Java 25 或更高版本*。因为使用了移植版的 Voxy，请安装 Java 25，并在启动器内将整合包的游戏 Java 设置为 JDK 25 或更高版本。如果不想使用 Java 25，可参考Q4的方法删除 Voxy。你可以在QQ群“FBlaze Matrix·Players”（右上角图标）的群文件中找到 JDK 25 的安装包，或者在 [123云盘](https://1835134308.share.123pan.cn/123pan/PpdmTd-HtTRH?pwd=fmsv#) 下载。

---

**Q2:** 我使用的是 AMD 显卡。

**A2:** 参考Q4的方法*删除 Voxy 模组*。

---

**Q3:** 我在使用过程中遇到Bug了，该如何反馈？

**A3:** 请*联系 [FeatherBlaze](https://docs.fblaze62.top/#%E7%AE%A1%E7%90%86%E5%91%98%E5%9B%A2%E9%98%9F)* 并发送版本文件夹中的`/logs/latest.log`和`/crash-reports/crash-yyyy-MM-dd_HH.mm.ss-client.txt`(如果有)。

---

**Q4:** 如何移除 Voxy 模组？

**A4:** 删除后*需要使用修改版本的 Voxy WorldGen*（此模组必加）。你可以在QQ群“FBlaze Matrix·Players”（右上角图标）的群文件中找到修改版本的 Voxy WorldGen，或者在 [123云盘](https://1835134308.share.123pan.cn/123pan/PpdmTd-HtTRH?pwd=fmsv#) 下载。

---

**Q5:** 我可以修改该整合包吗？

**A5:** *不行*。该整合包是版权所有的。

### 已知问题

Voxy 不兼容 AMD 显卡，可删除 Voxy并使用修改版本的 Voxy WorldGen（此模组必加）（参考[FAQ](https://docs.fblaze62.top/guide/player/download-modpack.html#faq)的Q4）。

Voxy 仅兼容 Java 25，请安装 Java 25，并在启动器内将整合包的游戏 Java 设置为 Java 25 或更高版本。如果不想使用 Java 25，可参考[FAQ](https://docs.fblaze62.top/guide/player/download-modpack.html#faq)中Q4的方法删除 Voxy。

部分优化模组无法添加，如光照引擎优化模组 ScalableLux（与 机械动力：航空学 的前置 Sable 不兼容）、加速渲染（与 Axiom 不兼容）等。
