>详细说明：[官方维基](https://mvplugins.org/core/)

1. 创建世界：`/mv create {NAME} {ENV} -s [SEED]`

   参数说明：{ }为必填参数，[ ]为选填参数。

    `{NAME}`：世界名称<br>
    `{ENV}`：世界维度（normal/NETHER/END）<br>
    `[SEED]`：种子或世界类型（flat/largebiomes）
2. 删除世界：`/mv delete <世界名>`，再输入`/mv confirm`确认
3. 导入已经创建好的世界：`/mv import <世界名> normal`
4. 传送到世界：传送自己使用`/mv tp <世界名>`，传送指定玩家使用`/mv tp <玩家名> <世界名>`
5. 设置世界出生点：先站在要设置的出生点位置，然后输入`/mv setspawn`
6. 传送回世界出生点：`/mv spawn`
7. 清除世界实体：清除全部使用`/mv purge <世界名> ALL`，清除动物使用`/mv purge <世界名> animals`，清除怪物使用`/mv purge <世界名> monsters`（`<世界名>`使用`ALL`选定所有世界）
