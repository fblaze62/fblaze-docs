详细说明：[Minecraft插件百科](https://mineplugin.org/LuckPerms)

Essentials插件使用说明：[Minecraft插件百科](https://mineplugin.org/Essentials)

1. 允许指定玩家使用LuckPerms相关指令：`/luckperms user <玩家名> permission set luckperms.* true`
2. 创建权限组：`/luckperms creategroup <权限组名称>`
3. 设置权限组可用权限：`/luckperms group <权限组名称> permission set <权限> <true/false>`（用“*”指代任意字符）
4. 删除权限组权限：`/luckperms group <权限组名称> permission unset <权限>`
5. 添加玩家到权限组：`/luckperms user <玩家名> parent add <权限组名称>`（新玩家默认位于“default”权限组）
6. 将玩家移出权限组：`/luckperms user <玩家名> parent remove <权限组名称>`
7. 继承权限组权限：`/luckperms group <权限组名称> parent add <继承自>`
8. 移除继承权限组：`/luckperms group <权限组名称> parent remove <继承自>`
