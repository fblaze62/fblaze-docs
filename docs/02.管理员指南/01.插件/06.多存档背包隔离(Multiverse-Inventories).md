详细说明：[MCBBS](https://archives.mcbbs.co/read.php?tid=909834)

1. 创建/删除世界组：先输入`/mvinv group`，然后输入`create`/`delete`，最后输入`<世界组名称>`
2. 编辑世界组：先输入`/mvinv group`，再输入`edit`，然后依次输入`<世界名称>`，输入完成后输入`@`，最后输入`all@`
3. 显示世界或世界组信息：`/mvinv info <世界名/世界组名>`
4. 显示组列表：`/mvinv list`
